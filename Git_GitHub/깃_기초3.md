# Git


<br> 


## 깃 브랜치 생성

```bash
git branch 이름
```

<br >

## 깃 브랜치 목록 확인

```bash
git branch
```

<br >

## 깃 브랜치 이동

```powershell
git switch 이름
```

<br >

## 깃 브랜치 생성과 동시에 이동

- 기존의 git checkout -b 이름

```powershell
git switch -c 이름
```

<br >

## 브랜치 이름 병경

```powershell
git branch -m 기존이름 변경이름
```

<br >

## 브랜치 삭제

- 강제삭제는 -D를 이용

```powershell
git branch -d 이름
```

<br >

## 여러 브랜치의 정보

```powershell
git log --all --decorate --oneline --graph
```
